Mission Emission

View the visualization View it on Github

In this project, CO2 emissions that stem from nautical exports from Brazil in 2014 are visualized. The user can explore where the exports are shipped and which countries import what products from Brazil. Most importantly, the visualization displays how much carbon dioxide is emitted for each of the product categories.

UniVRse

This was used in a study where we evaluated controller based steering or head movement based steering of the ship that is the best considering efficiency and effectiveness in a VR environment. Using the head to steer was better regarding these two things and it caused less motion sickness than steering with the controller.

Spotify Groups

In this project, the goal was to make a high fidelity prototype, and not a fully functional product. We decided to add a group feature to Spotify, where you can share your music in a more private way. We also added a party function, where you can add songs to the play queue without being the one who actually plays the music. The prototype is made with JustInMind.

Trump or Dump

Try the game View it on Github

This project is coded in HTML and JavaScript. We created a Markov-chain and made a game where the player should guess if a tweet from Donald Trump is a real tweet or a creation of our Markov-chain (which takes parts multiple real tweets and mix it into one).
